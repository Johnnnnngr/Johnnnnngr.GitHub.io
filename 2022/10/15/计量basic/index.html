计量basic

Vector Notation

向量的定义见矩阵basic

一些概念

欧几里得范数(Euclidean norm)

欧几里得范数,向量的欧几里得长度,即\(||x||=(\sum x_{i}^{2})^{1/2}=(x'x)^{1/2}\)(x是列向量).

Multivariate random vector

直译为多元随机向量..记作:\(X=\begin{bmatrix}X_{1} \\X_{2} \\.. \\X_{n} \end{bmatrix}\). 向量的元素就是一维的随机变量\(X_{i}\).

随机向量有一些性质:

  1. \(X=x\)代表了\(\{X_{1}=x_{1},X_{2}=x_{2}..X_{n}=x_{n}\}\).因此离散形式(连续形式也是类似的)下\[P(X=x)=P(X_{1}=x_{1},X_{2}=x_{2}..X_{n}=x_{n})=\prod\pi(X_{i}=x_{i})\].
  2. 同理,\(X<x\)代表了\(\{X_{1}<x_{1},X_{2}<x_{2}..X_{n}<x_{n}\}\).
  3. 该向量有联合分布函数\(f(x)=f(x_{1},x_{2}..x_{n})\), 其cdf为\[\int f(x)dx=\int..\int f(x_{1}..x_{n})dx_{1}..dx_{n}\]. 一般用向量x指代\((x_{1}..x_{n})\)构成的有序数组,除非要专门研究某维度的随机变量,不然都写成向量x.

三角不等式

这部分是为了让之后的证明简单一些所做的prerequisite.

  1. \(\forall x_{j}\in R\), \(|\sum x_{j}|\le \sum |x_{j}|\).

    证明: 非常经典的三角不等式. 即证:\(|x_{1}+..+x_{n}|\le |x_{1}|+..+|x_{n}|\). 将\(x_{1}\)到\(x_{n}\)重新排序为\(x_{j_{1}}..x_{j_{n}}\), 令\(\forall j_{i} \le j_{i+1}\), 有\(x_{j_{i}} \le x_{j_{i+1}}\), 即正序.则如果\(\exists j_{s}\)使得\(x_{j_{s}}\le 0\), 有$|x_{j_{s}}+A||x_{j_{s}}|+|A|, $, \(\forall A>0\).则令A为排除小于0的部分的子序,得证原不等式. Bruce是用\[-|x_{1}|\le x_{1}\le|x_{1}|\to -|x_{1}|+-|x_{2}|\le x_{1}+x_{2}\le |x_{1}|+|x_{2}|\],再推广至m维三角不等式做的,思路一样.

  2. for any vector \(x = (x_{1},..x_{m})'\), 有\(||x||\le \sum |x_{i}|\).

    证明: \(||x||=(\sum x_{i}^{2})^{1/2}\), 取平方得到\(\sum x_{i}^{2}\), 等式右边平方得到\(\sum |x_{i}|^{2}+2\sum |x_{i}||x_{j}|\).易得结果. Bruce是将向量特殊化, 即\(\sum |x_{i}|=1\), 则有\(||x||=\sum x_{i}^{2}\le1\)得证.

  3. Schwarz Inequality

    for any m维向量x和y, 有:\(|x'y|\le ||x||\cdot||y||\).

    证明: 即证\(|\sum x_{i}y_{i}|\le (\sum x_{i}^{2}\sum y_{i}^{2})^{1/2}\). 等式左边\(\le \sum|x_{i}y_{i}|=\sum|x_{i}||y_{i}|\le \frac{1}{2} (\sum x_{i}^{2}+\sum y_{i}^{2})\). 利用均衡不等式\((xy)^{1/2}\le \frac{1}{2}(x+y)\)(易证),得证.

  4. for any m维向量x和y,有: \(||x+y||\le ||x||+||y||\).

    证明: 感觉画一下图会秒杀,但是我们还是务实地用代数方法. 直接抄Bruce了 : \[||x+y||^{2}\le ||x||^{2}+2||x||\cdot ||y||+||y||^{2}=(||x||+||y||)^{2}\].得证.

随机向量的其他性质

m维随机向量是m个sample space组成的大sample space, 到m维欧几里得空间\(R^{m}\)的映射. 有以下其他性质:

  1. 随机向量的期望是每个一维变量期望组成的m维向量: \(E[X]=\begin{pmatrix}E[X_{1}] \\E[X_{2}] \\... \\E[X_{m}] \end{pmatrix}=\begin{pmatrix}\mu _{1} \\\mu _{2} \\... \\\mu _{m} \end{pmatrix}=\mu\)

  2. 随机向量的variance 为\(Var[X]=E[(X-E[X])(X-E[X])']=\sum\). (课内将其记作\(\Omega\)).

    \(Var[X]\)是一个方阵,为\(\begin{pmatrix} \sigma _{1}^{2} &\sigma _{12} &... &\sigma _{1m} \\ \sigma _{21} &\sigma _{2}^{2} &... &\sigma _{2m} \\ ... &... &... &... \\ \sigma _{m1} &\sigma _{m2} &... &\sigma _{m}^{2} \end{pmatrix}\).对角线是单随机变量的方差, 其他位置是两个随机变量的协方差. \(\sum\)有性质如下:

    1. 对称, 即\(\sum =\sum '\). 易证

    2. positive semi-definite. 证明: \(\forall m\times 1\)向量a,有: \[a'\sum a=a'E[(X-E[X])(X-E[X])']a=E[a'(X-E[X])(X-E[X])'a]=E[z^{2}]\].

      其中\(z=a'(X-E[X]) \ge 0\).得证

    3. 如果\(X\in R^{m}\), \(\mu\)为mx1的向量,\(sum\)为mxm的矩阵,A为\(q\times m\)的矩阵, 则随机变量AX的期望向量为\(A\mu\), 方差矩阵为\(A\sum A'\). 自证,不难

    4. 如果\(X\in R^{m}\), 则\(E||X||\le \infty\)等价于\(E|X_{j}|\le\infty,\forall j \in Domain\).

      证明: if : \(E||X||=E[(\sum X_{i}^{2})^{1/2}]\le E[\sum |X_{i}|]=\sum E|X_{i}|\le mC<\infty\).

      ​ only if: 用到定理: 向量的模必不小于其分量. 则\(||X||\ge |X_{i}|\to \infty>E||X||\ge E|X_{i}|\).
