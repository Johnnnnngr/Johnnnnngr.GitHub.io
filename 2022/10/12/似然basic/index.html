最大似然估计

来源: BRUCE E. HANSEN

Parametric Model

Definition: 随机变量X(假定为iid)的parametric model是一个依赖于未知参数$ $的完全概率函数. (严格定义: A PM for a random sample is the assumption that \(X_i\), i = 1,..,n, are i.i.d. with known density function \(f(x|\theta)\) or mass function \(\pi(x|\theta)\) with unknown parameter \(\theta \in \Theta\)).

注:其中$ $一般是scalar,但也可以是一个向量(因此实际上会有很多未知一维变量,过于复杂,我们仅考虑一维未知量)

注: 实际上可以不是iid,但是我们仅考虑iid.

离散形式下我们可以将PM写作一个概率密度函数\(\pi(X|\theta)\),连续形势下我们可以将其写作\(f(x|\theta)\). 令未知参数\(\theta \in \Theta\),后者称为parameter space. 因为PM的未知参数只能从一个特定集合\(\Theta\)中选取,所以规定\(\Theta\)中所有参数构成的所有分布函数组成的集合称为parametric family. eg: 假定一个PM为 \(X\sim N(\mu,\sigma^2 )\),其密度函数为\(f(x|\mu,\sigma^2)\),有参数\(\mu\in R,\sigma^2 \in(0,\infty )\).

注: PM可以是自己随便构造的.

Definition: 当\(\exists \theta_i \in \Theta\)使得\(f(x|\theta_i)=f(x)\),后者即true data distribution.而且\(\theta_i\)的值唯一,则称该\(\theta_i\)为true parameter value, 该PM is correctly specified.

反之,如果不存在参数使得模型可以反应真实分布,则称该PM is mis-specified.

eg: 对于真实分布\(f(x)=2exp(-2x)\), 指数模型\(f(x|\lambda)=\lambda^{-1}exp(\frac{-x}{\lambda } )\)是一个correctly specific model, 因为仅存在\(\lambda_0=0.5\)使模型符合真实分布, 但指数正态分布就是mis-specified model了,因为不存在满足要求的参数使该模型符合真实分布.

Likelihood

基本假设: 我们所用的模型是correctly specified的.

Definition: 似然是用指定PM计算的所有实验的联合分布密度(The likelihood is the joint density of the observations calculated using the model.)

因为我们假定随机变量是iid,所以其联合分布密度就是各个分布的累积. 同时我们仅用一个model, 所以每次实验的分布都是一样的.因此似然函数(likelihood function)等于如下函数:\[L_n(\theta)\equiv f(x_1,..x_n|\theta)=\prod f(x_i|\theta).\]

其中\(L_n\)指似然函数.

类似于概率函数分析哪个值最可能发生, 我们用似然函数分析参数取哪个值最可能符合我们的观察.因此, 最大化似然的参数\(\hat{\theta }\)就是实际参数(或最接近的,但我们已经假设用的模型是correct specified的,所以一般就是).

由此引出最大似然估计

Maximum Likelihood estimator

\(\hat{\theta }\)就是最大化\(L_n(\theta)\)的参数,即:\(\hat{\theta }=\underset{\theta \in \Theta}{argmax} L_n(\theta)\)

例子自己看例题. 求\(\hat{\theta }\)一般就是FOC+SOC两步走.

为了简便计算,我们一般取似然函数的log: \(\mathfrak{l} _n(\theta)\equiv log(L_n(\theta))\).

可行性证明: 因为取自然对数不会改变原函数在各个\(\theta\)的单调性,即其导数符号处处相同,所以可用.

Likelihood Analog Principle

定义expected log density: \(\mathfrak{l} (\theta)=E[logf(X|\theta)]\). 当模型是correctly specified时, 实际的参数\(\theta_0\)最大化了这个函数(自己证明).

eg: \(f(x|\lambda)=\lambda^{-1}exp(-x/\lambda)\), 则其expected log density为: \[\mathfrak{l} (\lambda)=E[logf(X|\lambda)]=E[-log\lambda-X/\lambda]=-log\lambda-E[X]/\lambda=-log\lambda-\lambda_0/\lambda\]. FOC:\(-\lambda^{-1}+\lambda_0\lambda^{-2}=0\).

有唯一解\(\lambda=\lambda_0\).

注意此处的density是对全体X,即population而言的.

类似的,我们对\(\mathfrak{l} (\theta)\)进行sample analog:

\[\overline { \mathfrak{l}}_{n} ( \theta )= \frac{1}{n} \mathfrak{l}_{n} ( \theta )= \frac{1}{n} \sum logf(X_i | \theta ) \].

同理, \(\hat{\theta}\) 使它最大化.

Invariance Property

if \(\hat{\theta}\)是MLE of \(\theta \in R^m\), 那么对于任何变换\(\beta=h(\theta)\in R^l\), 他的MLE,即\(\hat{\beta}=h(\hat{\theta})\). 自己证明.

Likelihood Ratio Test for simple Hypothesis

仅考虑最简单的\(H_0\)和\(H_1\):

\(\left\{\begin{matrix}H_0:\theta=\theta_0 \\H_1:\theta = \theta_1 \end{matrix}\right.\)

如果不接受\(H_0\),只要使\(L_n(\theta_1)\)在一定程度上比\(L_n(\theta_0)\)大,即\(\theta_1\)比\(\theta_0\)更有可能.

关键词: 一定程度,大. 所以我们可以用比例形式判断: \[L= \frac{L_{n}( \theta_{1} )}{L_{n}( \theta_{0} )}\].

此时,如果 \(L \le c\) 则accept \(H_0\), 如果\(L \ge c\)则reject \(H_0\).

为了方便计算,我们采用log形式,则定义likelihood ratio statistic为: \(LR_n=2(\mathfrak{l} _n(\theta_1)-\mathfrak{l}_n(\theta_0))\). \(\to\) if \(LR_n \le c\)则接受\(H_0\), if \(LR_n\ge c\)则拒绝\(H_0\). 至于为什么要乘2,一部分是历史原因,另一部分是因为这样在比较critical value 的时候会方便,而且乘2对结果其实没什么影响.

Likelihood Ratio Test against composite alternatives

\(\left\{\begin{matrix}H_0: \theta =\theta_0 \\H_1:\theta \ne \theta_0 \end{matrix}\right.\)

令\(\hat{\theta}\)为MLE,则此时的LR为\(LR_n=2(\mathfrak{l_n(\widehat{\theta} )-l_n(\theta_0)} )\). 此时判别式和上面一样.

t-test

simple null(单值)against composite alternatives(区间)的ratio类似于t-test. 利用泰勒展开证明.

令\(\theta \in R^k\), \(LR_n=2(\mathfrak{l}_n (\hat{\theta})-\mathfrak{l}_n (\theta_0))\). 以\(\hat\theta\)为起点对\(\mathfrak{l}_n (\theta_0)\)做二阶泰勒展开,得:

\[\mathfrak{l}_{n} ( \theta_{0} ) \simeq \mathfrak{l}_{n} ( \widehat{\theta} )+ \frac{\partial \mathfrak{l}_{n} ( \widehat{\theta} )'( \hat{\theta}-\theta_{0})}{\partial \theta}+\frac{1}{2}(\hat{\theta}-\theta_{0})' \frac{\partial^2}{\partial \theta\partial \theta'} \mathfrak{l}_{n} (\widehat{\theta} )(\hat{\theta}-\theta_{0}) \].

我们在求\(\hat\theta\)的时候已经使$=0 $了, 而他的SOC是Hessian \(H_f\) = \(-\widehat{V} ^{-1}\). 后者是Hessian Estimator of asymptotic variance of \(\hat\theta\).

得到: \(\mathfrak{l}_n (\theta_0)\simeq\mathfrak{l}_n (\widehat{\theta} )-\frac{1}{2}(\hat\theta-\theta_0)'\widehat{V} ^{-1}(\hat\theta-\theta_0)\).

得到: \(LR_n\simeq(\hat\theta-\theta_0)'\widehat{V} ^{-1}(\hat\theta-\theta_0)\). 当\(n\to\infty\), \(LR_n \underset{d}{\to} X_m^2\). m是\(\theta\)的维数(向量长度)(这个自己证明吧).

则: \(c=q_{1-\alpha}\). 后者为卡方分布的(\(1-\alpha\)) 区间.

此时,在一维情况下\((\hat\theta-\theta_0)'\widehat{V} ^{-1}(\hat\theta-\theta_0)=T^2\), 则在大数和一维下,可用\(|T|>c\)代替\(LR_n>c\).

注: 以上情况仅在大数时成立,若样本小则LR和X的差距很大.
